# ✅ To-Do List

A clean and minimal **To-Do List** web app built with **HTML, CSS, and JavaScript**. Add tasks, mark them complete, and delete them — all without any backend or framework.

## 🎮 How to Use

1. Type a task in the input field
2. Press **Enter** to add it to the list
3. Click a task to mark it as **done** ✔️
4. Click the **delete icon** to remove a task

## ✨ Features

- ➕ **Add Tasks** — Type and press Enter to add instantly
- ✔️ **Mark Complete** — Click any task to toggle done state
- 🗑️ **Delete Tasks** — Remove individual tasks with one click
- 🎨 **Custom Styling** — Clean UI with custom CSS
- 🔤 **Font Awesome Icons** — Delete icons via Font Awesome CDN
- ⚡ **No Dependencies** — Pure HTML, CSS, and JS

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Page structure |
| CSS3 | Custom styling (`custom.css`) |
| Vanilla JavaScript | Task add / delete / toggle logic |
| Font Awesome | Icons (via CDN) |

## 🚀 How to Run

No setup needed — it's a static web app!

```bash
git clone https://github.com/anshuman-gkp1/To-do-list.git
cd To-do-list
```

Open `to-do-list/index.html` in any web browser.
